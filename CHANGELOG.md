Changelog
 
All notable changes to this project will be documented in this file.
 
[1.0.0] - 2026-01-30
 
Added
 
- Initial release of Aprit, built with Qt6 for Linux systems
- Core application framework with native Linux desktop integration
- Basic tabbed interface with initial layout logic
- Fundamental network connection and file download functionality
- Multi-language foundation with core language support
- Single-file executable packaging for simplified distribution
